Answer to the Quesnion no 1:
    var: is function-scoped and allows redeclaration and reassignment.
    let is block-scoped and allows reassignment but not redeclaration in the same scope.
    const is block-scoped like let, but it does not allow reassignment after initialization.

Answer to the Question no 2:
    The spread operator (...) is used to expand elements of an iterable or properties of an object into individual elements.

Answer to the Question no 3:
    map() creates a new array by applying a transformation function to each element.
    filter() creates a new array containing only elements that satisfy a given condition.
    forEach() executes a function on each element but does not return a new array.

Answer to the Question no 4:
    An arrow function is a shorter syntax for writing functions introduced in ES6.

Answer to the Question no 5:
    Template literals are a way to define strings using backticks (`) instead of quotes.
